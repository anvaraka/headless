## Intro

The `remove` command is used to clean up your project.

**Using this command to remove components and their associated files is very powerful and often irreverisble - Please use with care**

```bash copy
umi remove
```

## Options

| Option | Description | Default |
| --- | --- | --- |
| `componentName` | The name of the component you are looking to clean up. | None |
